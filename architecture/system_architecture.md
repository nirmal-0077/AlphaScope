# AlphaScope System Architecture

## Overview

AlphaScope is an AI-Powered Multi-Agent Market Intelligence and Decision Support Platform.

The platform consists of:

* Frontend (Next.js)
* Backend (FastAPI)
* PostgreSQL Database
* ChromaDB Vector Database
* Multi-Agent Intelligence Layer
* RAG Chat Layer
