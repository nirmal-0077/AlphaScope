# AlphaScope

Multi-Agent Market Intelligence and Business Research Platform

## Vision

AlphaScope automates company, industry, and competitor research through specialized AI agents that investigate financial filings, news, sentiment, risks, and trends, generating evidence-backed reports and conversational insights.

## Tech Stack

Frontend:

* Next.js

Backend:

* FastAPI

Database:

* PostgreSQL

Vector Database:

* ChromaDB

AI:

* OpenAI / Claude / Gemini
